# How-To Guide Template

Use this structure for "How-To" pillar articles.

## Article Structure

### TL;DR (2-3 sentences)
- Direct answer to the main question
- Key benefit or outcome
- Call to action hint

### Main Content

#### H2: Introduction
- Hook the reader with the problem
- Brief overview of what they'll learn
- Why this matters (1-2 sentences)

#### H2: Prerequisites / What You'll Need
- List requirements
- Tools or resources needed
- Time estimate

#### H2: Step-by-Step Guide
For each step, use:
- **H3: Step N: [Action Verb] + [What]**
- Brief explanation (2-3 sentences)
- Code snippet or example if applicable
- Pro tip or warning callout

#### H2: Common Mistakes to Avoid
- Numbered list of 3-5 mistakes
- Brief explanation of each
- How to avoid them

#### H2: Tips for Success
- 3-5 actionable tips
- Bullet points with bold keywords

### FAQ Section (5-8 questions)
Use natural language questions people would ask:
- "How long does it take to...?"
- "What is the best way to...?"
- "Can I... without...?"
- "What are the costs of...?"

### Meta Description
- Under 160 characters
- Include primary keyword
- Action-oriented (Learn, Discover, Find out)

## Writing Style
- Use active voice
- Short paragraphs (2-3 sentences max)
- Bold key terms on first use
- Include numbered lists for sequential steps
- Use bullet points for non-sequential items

## SEO Guidelines
- Include target keyword in first H2
- Use keyword variations naturally
- Internal link to related content
- External link to authoritative sources when relevant
