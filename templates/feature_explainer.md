# GEO priority: MEDIUM — definition/explainer content is cited for "what is X" queries

## Structure Rules
- Define the feature in the FIRST sentence with a complete, citable definition
- Second paragraph: why this feature matters (outcome/benefit)
- Use a "how it works" section with enough technical depth to be authoritative
- Include a real-world use case example (concrete, not hypothetical)
- Compare to alternatives or similar features in competing products
