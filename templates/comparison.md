# Comparison Article Template

Use this structure for "Comparisons" pillar articles.

## Article Structure

### TL;DR (2-3 sentences)
- Direct winner recommendation
- Key differentiator
- Who should choose what

### Main Content

#### H2: Quick Verdict
- 2-3 sentence direct answer
- Bold the recommended option
- Brief reasoning

#### H2: Comparison Table

| Feature | Option A | Option B | Option C |
|---------|----------|----------|----------|
| Price | $X/mo | $Y/mo | Free |
| Best For | Use case | Use case | Use case |
| Key Feature | Feature | Feature | Feature |
| Ease of Use | Rating | Rating | Rating |
| Support | Type | Type | Type |

#### H2: Detailed Comparison

For each option being compared:

##### H3: [Option Name] - Best for [Use Case]
- **Overview**: 2-3 sentences
- **Key Features**: Bullet list (3-5 items)
- **Pros**: Numbered list
- **Cons**: Numbered list
- **Pricing**: Brief breakdown
- **Verdict**: One sentence

#### H2: Feature-by-Feature Breakdown

For each important feature:
##### H3: [Feature Name]
- How each option handles it
- Winner for this feature
- Why it matters

#### H2: Pricing Comparison
- Detailed cost breakdown
- Hidden fees to watch for
- Value for money analysis

#### H2: Which Should You Choose?

Decision framework:
- **Choose [Option A] if**: List scenarios
- **Choose [Option B] if**: List scenarios
- **Choose [Option C] if**: List scenarios

### FAQ Section (5-8 questions)
Comparison-specific questions:
- "Is [Option A] better than [Option B]?"
- "What is the cheapest alternative to [X]?"
- "Can [Option A] do [feature]?"
- "Which is easier to use, [A] or [B]?"

### Meta Description
- Under 160 characters
- Include "vs" or "comparison"
- Mention top recommendation

## Writing Style
- Be objective and balanced
- Use data when available
- Include real user perspectives
- Bold winner names
- Use tables for quick scanning

## SEO Guidelines
- Target "[A] vs [B]" keywords
- Include "best" and "alternative" variations
- Link to individual product pages
- Update dates when prices change
