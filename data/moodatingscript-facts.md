## Core Features

### Profiles & Discovery
- Advanced user profiles: name, age, gender, orientation, bio, photo & video, smart photo ordering, job & education fields
- Swipe-based discovery (Like / Pass / Match when both users like)
- Advanced search & filters: age range, gender preference, location (country/city), lifestyle tags
- Popularity ranking & activity-based recommendations

### Matching & Algorithms
- AI-powered matching algorithm (v1.7+)
- Location-based matching
- Interest similarity signals
- Intelligent match engine based on behavior, preferences, and activity

### Messaging & Interaction
- Real-time 1–1 chat
- Emoji & GIF support
- Photo sharing in chat
- Message reactions
- Virtual gifts (send between users)

### Monetization & Payments
- Monthly subscription payment system
- Wallet-based credits system
- Profile boost & visibility upgrades
- Boost Profile Using Story
- Payment gateways: **CCbill, Stripe, PayPal**

### Trust & Safety
- Phone (SMS) verification
- Blue check verification badge
- Safety center & reporting tools
- Block contacts feature
- GDPR compliance — consent management, data access controls, privacy-focused

### Mobile & UX
- PWA (Progressive Web App) for iOS and Android
- Dark mode
- Responsive design (mobile-optimized)
- In-app notifications & alerts

### Admin & Operations
- Full admin control panel: manage users, subscriptions, revenues
- Profile seeding (demo profiles for launch)
- Multi-language support
- AI-powered profile management (v1.7+)
